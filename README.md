# AANGAN – Traditional Dining Experience

AANGAN is a restaurant website inspired by the warmth of a traditional Indian courtyard and the rich culinary heritage of Maharashtra. The project was created to provide an elegant and welcoming online presence for a restaurant that values authentic flavors, family gatherings, and genuine hospitality.

The website features a heritage-inspired design, a detailed menu showcase, an image gallery, an about section highlighting the restaurant's story, and a contact section for reservations and enquiries.
